# LocalShop Agent

AI-powered shopping assistant using Algolia Agent Studio, LLMs, and WhatsApp integration.

Tech Stack:
- Django
- PostgreSQL
- Algolia
- Agent Studio
- LLM (OpenAI/Gemini)
- WhatsApp API
- Next.js
